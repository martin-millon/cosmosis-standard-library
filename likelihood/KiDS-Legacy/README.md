This folder contains a fits file with the following extensions:
- En: COSEBIs E modes
- COVMAT: Covariance matrix of COSEBIs E modes
- NZ_SOURCE: Redshift distribution per tomographic bin

Additionally, we provide the dz covariance matrix in ascii format and the covariance matrix of IA parameters (see Sect. 2.4.4 and Appendix B in [Wright et al. 2025](https://arxiv.org/abs/2503.19441))

Note: We provide 6 COSEBI E modes here, which is the fiducial number in the KiDS-Legacy cosmic shear analysis. For additional modes and other summary statistics, see the data products on the KiDS-Website: https://kids.strw.leidenuniv.nl/sciencedata.php
